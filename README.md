# exoplanet-analysis

An analysis of exoplanet discovery trends over the last 20 years, using data from the NASA Exoplanet Archive.

The notebook walks through how exoplanet discovery has been impacted over the years, noteably by the Kepler, TESS, and JWST launches.

## Data

Sourced from the [NASA Exoplanet Archive](https://exoplanetarchive.ipac.caltech.edu/), querying the Planetary Systems table.

## Tools

Python, pandas, matplotlib, seaborn
